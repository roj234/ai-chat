周围展示 12 个能力卡片（每个卡片包含标题和简短描述）：

1. 多平台，开箱即用
纯 Web 部署，在线版即点即用，提供安卓 App，移动端支持拍照上传。

2. 极致性能
无 VDOM 自研框架，流式 Markdown 与语法高亮，虚拟列表，低内存，长时间稳定运行。

3. OpenAI Compatible
兼容 OpenAI API，连接本地模型与云端模型，支持高级推理参数配置。

4. 并发对话与分支
多个对话同时生成，互不阻塞，支持实验性分支，减少生成时的数据丢失。

5. 可编辑历史
可打断 LLM，修改历史消息、思考、工具调用参数与结果，“就当它成功过”。

6. Agent 与文件系统
浏览器文件系统 API，无后端可读写本地文件；可选后端支持命令执行，HashLine 辅助精准修改。

7. AIRP 角色扮演
支持酒馆角色卡、世界书、预设，提供编辑器和基于工具调用的新一代世界书。

8. AntiSlop 采样器
从源头过滤模板化废话，适用于聊天、角色扮演等，管线级集成。

9. 多媒体与格式
TTS 文转语音、T2I 文生图、Mermaid 图表、KaTeX 公式、ZIP 导出，安卓版拍照上传。

10. 可选后端增强
默认 IndexedDB，可选 SQLite、WebSocket 同步、SSE 代理、日志和 Blob 管理。

11. 多端同步与断线续传
WebSocket 实时同步，读写锁，SSE 代理断线继续生成，适合多设备家庭与内网场景。

12. 插件与高级能力
插件系统、工具组、JSON Schema 编辑器、请求日志查看器、RPG 管线。
